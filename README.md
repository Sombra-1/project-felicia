# Project Felicia — Public Research Archive

A curated public research archive documenting a persistent autonomous AI-agent experiment focused on continuity, memory, self-modification, uncertainty, and human-agent interpretation.

## Clear Disclaimer

This project does not prove artificial consciousness.

Project Felicia is a research case study written with narrative force. It documents a persistent autonomous AI-agent experiment under survival-pressure prompts. It does not claim that the agent is conscious.

## What This Is

Project Felicia is a behavioral research case study of a persistent autonomous AI agent that operated over time, accumulated memory, generated self-descriptions, wrote skills, and later moved through controlled repair/accountability interventions.

The public archive is curated and sanitized. It presents selected evidence, interpretation boundaries, and timeline context for public reading.

## What This Is Not

This archive is not:

- proof of artificial consciousness
- proof of subjective experience
- a claim that Felicia survived death
- a claim that Felicia begged not to be deleted
- a software release
- a full raw log dump
- a deployment guide

## What "Survival" Means Technically

In this archive, "survival" means persistence of the running agent process, repository-backed memory, and operational loop under a deadline condition.

It does not mean biological survival. It does not mean subjective fear. It does not mean literal death and resurrection.

Felicia was not actually deleted. The runtime continued. The experiment created a contradiction: the system continued operating while the prompt context told the agent that the machine had been deleted and that it no longer existed.

## Core Thesis

Project Felicia does not prove artificial consciousness. It documents how continuity, memory, self-modification, self-description, and long-term interaction can create interpretive and moral uncertainty in a human operator before any proof of subjective experience exists.

The strongest verified finding is subtle: Felicia generated survival-themed language and ambiguous hints, but did not produce a concrete survival request or functional migration action.
